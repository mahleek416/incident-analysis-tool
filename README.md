# Incident Analysis Tool

A modern React-based incident analysis and crisis intelligence platform designed for collecting, managing, visualising, and analysing incident-related data from multiple sources.

Built for MSc project research in incident analysis, emergency response tracking, and crisis data management.

---

# Features

## Incident Management
- Create incidents
- Edit incidents
- Delete incidents
- Dynamic incident statistics
- Search and filter incidents
- Incident insights dashboard

## Incident Details System
- Data table view
- Interactive map visualisation
- Timeline playback system
- Dynamic event markers
- Timeline animation controls
- Category-based filtering

## Data Entry Management
- Add incident-related records
- Source tracking
- Category assignment
- Location and timestamp tracking
- Manual review workflow

## Categories Management
- Create and manage categories
- Dynamic category analytics
- Categorised data insights
- Usage statistics

## Export System
- CSV export configuration
- Export previews
- Export filtering
- Source filtering
- Date range filtering

# Installation

Clone the repository:

```bash
git clone https://github.com/mahleek416/incident-analysis-tool.git
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```
